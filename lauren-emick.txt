    37. Walk me through Lambda's git flow. 
        1. Fork 
        2. Add TL as a collaborator
        3. Clone
        4. In CLI, use cd to go to desired directory
        5. git clone <link-to-repo> command
        6. cd into the newly created repo using the cd <repo-name> command
        7. code . and then work on your repo
        8. git checkout -b 'firstname-lastname' command to create your own branch name for the repo
        9. git add . will add the repo changes and make them ready for committing
        10. git commit -m "Your helpful commit message here"
        11. git push -u origin branch-name
        12. go to your forked project on github and change branch to your name-branch
        13. click the New pull request button 
        14. When you get to the PR page, make sure to click the dropdown button that says base repository and CHANGE it to be your FORKED REPO 
        15. Leave comments, add your TL to the Reviewers, and then submit the pull request

Stretch Questions

    1. What is the difference between an inline element and a block element?
    Inline elements take up only as much space as is needed, block elements take up an entire row

    2. What happens when an element is positioned absolutely? 
    When you absolute position an element, it no longer takes up the space it used to and is now essentially ‘floating’ on your screen, above the surrounding elements. You can now position an absolute value exactly where you want it to go.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    position: relative; 
  
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    Block: div, h1-h6, p, ul/ol, li
    Inline: span, strong 
    Inline-block: img

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
    A box that wraps around each element and helps create your website layout. 
    Total element width = width + left padding + right padding + left border + right border + left margin + right margin
    Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
